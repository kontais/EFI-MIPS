NAME

     toascii - convert a byte to 7-bit ASCII

SYNOPSIS

     #include <ctype.h>

     int
     toascii(int c)

DESCRIPTION

     The toascii() function strips all but the low 7 bits from a letter, in-
     cluding parity or other marker bits.

RETURN VALUES

     The toascii() function always returns a valid ASCII character.
