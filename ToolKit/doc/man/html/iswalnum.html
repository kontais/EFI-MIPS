NAME

     iswalnum - alphanumeric wide character test

SYNOPSIS

     #include <wctype.h>

     int
     iswalnum(wint_t wc)

DESCRIPTION

     The iswalnum() function tests for any wide character for which iswalpha(3) or
     iswdigit(3) is true.  In the ASCII character set, this includes the fol-
     lowing characters:

     060 ``0''	   061 ``1''	 062 ``2''     063 ``3''     064 ``4''
     065 ``5''	   066 ``6''	 067 ``7''     070 ``8''     071 ``9''
     101 ``A''	   102 ``B''	 103 ``C''     104 ``D''     105 ``E''
     106 ``F''	   107 ``G''	 110 ``H''     111 ``I''     112 ``J''
     113 ``K''	   114 ``L''	 115 ``M''     116 ``N''     117 ``O''
     120 ``P''	   121 ``Q''	 122 ``R''     123 ``S''     124 ``T''
     125 ``U''	   126 ``V''	 127 ``W''     130 ``X''     131 ``Y''
     132 ``Z''	   141 ``a''	 142 ``b''     143 ``c''     144 ``d''
     145 ``e''	   146 ``f''	 147 ``g''     150 ``h''     151 ``i''
     152 ``j''	   153 ``k''	 154 ``l''     155 ``m''     156 ``n''
     157 ``o''	   160 ``p''	 161 ``q''     162 ``r''     163 ``s''
     164 ``t''	   165 ``u''	 166 ``v''     167 ``w''     170 ``x''
     171 ``y''	   172 ``z''

RETURN VALUES

     The iswalnum() function returns zero if the wide character tests false and re-
     turns non-zero if the wide character tests true.
