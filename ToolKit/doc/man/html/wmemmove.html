NAME

     wmemmove - copy wchar_t string

SYNOPSIS

     #include <string.h>

     void *
     wmemmove(void *dst, const void *src, size_t len)

DESCRIPTION

     The wmemmove() function copies len wchar_t's from string src to string dst.
     The two strings may overlap; the copy is always done in a non-destructive
     manner.

RETURN VALUES

     The wmemmove() function returns the original value of dst.
