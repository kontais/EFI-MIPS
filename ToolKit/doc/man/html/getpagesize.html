NAME

     getpagesize - get system page size

SYNOPSIS

     #include <unistd.h>

     int
     getpagesize(void)

DESCRIPTION

     Getpagesize() returns the number of bytes in a page.  Page granularity is
     the granularity of many of the memory management calls.

     The page size is a system page size and may not be the same as the under-
     lying hardware page size.

SEE ALSO

     pagesize(1),  sbrk(2)

HISTORY

     The getpagesze() function call appeared in 4.2BSD.
