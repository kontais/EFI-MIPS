NAME

     InitializeLibC - initialize the EFI C library

SYNOPSIS

     #include <atk_libc.h>

     int
     InitializeLibC(EFI_HANDLE ImageHandle, EFI_SYSTEM_TABLE *SystemTable)

DESCRIPTION

     The InitializeLibC() function must be called before any other EFI 
     libc functions are called.
