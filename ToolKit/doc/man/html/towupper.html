NAME

     towupper - lower case to upper case letter conversion

SYNOPSIS

     #include <wctype.h>

     wint_t
     towupper(wint_t wc)

DESCRIPTION

     The towupper() function converts a lower-case letter to the corresponding
     upper-case letter.  If the argument is a lower-case letter, the towupper()
     function returns the corresponding upper-case letter if there is one;
     otherwise the argument is returned unchanged.
