NAME

     wcscmp, wcsncmp, - compare strings

SYNOPSIS

     #include <string.h>

     int
     wcscmp(const wchar_t *s1, const wchar_t *s2)

     int
     wcsncmp(const wchar_t *s1, const wchar_t *s2, size_t len)

DESCRIPTION

     The wcscmp() and wcsncmp() functions lexicographically compare the null-
     terminated strings s1 and s2.

RETURN VALUES

     The wcscmp() and wcsncmp() return an integer greater than, equal to, or
     less than 0, according as the string s1 is greater than, equal to, or
     less than the string s2.

     The wcsncmp() compares not more than len characters.
