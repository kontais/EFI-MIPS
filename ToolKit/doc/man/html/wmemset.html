NAME

     wmemset - write a wchar_t to wchar_t string

SYNOPSIS

     #include <string.h>

     void *
     wmemset(void *b, int c, size_t len)

DESCRIPTION

     The wmemset() function writes len wchar_t's of value c to the string b.

RETURNS

     The wmemset() function returns its first argument.

SEE ALSO

     bzero(3),	swab(3)
