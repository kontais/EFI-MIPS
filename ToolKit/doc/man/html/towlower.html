NAME

     towlower - upper case to lower case letter conversion

SYNOPSIS

     #include <wctype.h>

     wint_t
     towlower(wint_t wc)

DESCRIPTION

     The towlower() function converts an upper-case letter to the corresponding
     lower-case letter.

RETURN VALUES

     If the argument is an upper-case letter, the towlower() function returns
     the corresponding lower-case letter if there is one; otherwise the argu-
     ment is returned unchanged.
