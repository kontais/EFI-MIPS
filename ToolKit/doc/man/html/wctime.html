NAME

     wasctime, wctime, - transform binary date and time values

SYNOPSIS

     #include <wchar.h>
     #include <time.h>

     extern wchar_t *tzname[2];

     wchar_t *
     wctime(const time_t *clock)

     wchar_t *
     wasctime(const struct tm *tm)

DESCRIPTION

     The functions wctime(), takes as an argument a time value representing
     the time in seconds since the Epoch (00:00:00 UTC, January 1, 1970; see
     time(3)).

     The wctime() function adjusts the time value for the current time zone in
     the same manner as localtime(), and returns a pointer to a 26-character
     string of the form:

           Thu Nov 24 18:22:48 1986\n\0

     All the fields have constant width.

     The wasctime() function converts the broken down time in the structure tm
     pointed at by *tm to the form shown in the example above.

STANDARDS

     The wasctime(), wctime(), functions conform to ISO 9899: 1990 (``ISO
     C''), and conform to IEEE Std1003.1 (``POSIX'') provided the selected lo-
     cal timezone does not contain a leap-second table (see zic(8)).
