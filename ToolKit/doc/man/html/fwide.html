NAME

     fwide - determines the orientation of the stream

SYNOPSIS

     #include <wchar.h>

     int 
     fwide(FILE *stream, int mode)

DESCRIPTION

     The function determines the orientation of the stream stream. If mode is
     greater than zero, it first attempts to make the stream wide oriented. 
     If mode is less than zero, it first attempts to make the stream byte oriented.

RETURN VALUES

     In any event, the function returns:

     o  A value greater than zero if the stream is left wide-oriented. 
     o  Zero if the stream is left unbound. 
     o  A value less than zero if the stream is left byte-oriented.

STANDARDS

     The functions fwide() conforms to ISO 9899: 1990 (``ISO C'').
