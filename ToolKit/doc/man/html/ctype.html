NAME

     isalnum, isalpha, isascii, isblank, iscntrl, isdigit, isgraph, islower,
     isprint, ispunct, isspace, isupper, isxdigit, toascii tolower, toupper, -
     character classification macros

SYNOPSIS

     #include <ctype.h>

     int
     isalnum(int c)

     int
     isalpha(int c)

     int
     isascii(int c)

     int
     iscntrl(int c)

     int
     isdigit(int c)

     int
     isgraph(int c)

     int
     islower(int c)

     int
     isprint(int c)

     int
     ispunct(int c)

     int
     isspace(int c)

     int
     isupper(int c)

     int
     isxdigit(int c)

     int
     toascii(int c)

     int
     tolower(int c)

     int
     toupper(int c)

DESCRIPTION

     The above functions perform character tests and conversions on the inte-
     ger c. They are available as macros, defined in the include file
     <ctype.h>, or as true functions in the C library.	See the specific manu-
     al pages for more information.

SEE ALSO

     isalnum(3),  isalpha(3),  isascii(3),  isblank(3),  iscntrl(3),
     isdigit(3),  isgraph(3),  islower(3),  isprint(3),  ispunct(3),
     isspace(3),  isupper(3),  isxdigit(3),  toascii(3),  tolower(3),
     toupper(3),  ascii(7)

STANDARDS

     These functions, except for isblank(), toupper(), tolower() and
     toascii(), conform to ISO 9899: 1990 (``ISO C'').
