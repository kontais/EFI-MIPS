NAME

     tan, tanf - tangent functions

SYNOPSIS

     #include <math.h>

     double
     tan(double x)

     float
     tanf(float x)

DESCRIPTION

     The tan() and the tanf() functions compute the tangent of x (measured in
     radians).	A large magnitude argument may yield a result with little or
     no significance.  For a discussion of error due to roundoff, see math(3).

RETURN VALUES

     The tan() function returns the tangent value.

SEE ALSO

     acos(3),  asin(3),  atan(3),  atan2(3),  cos(3),  cosh(3),  math(3),
     sin(3),  sinh(3),	tanh(3)

STANDARDS

     The tan() function conforms to ISO 9899: 1990 (``ISO C'').
