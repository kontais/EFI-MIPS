_LIBC_EfiExit 	     Libc    Manual 	       LIBC_EfiExit

NAME

     _LIBC_EfiExit - perform normal program termination

SYNOPSIS

     #include <atk_libc.h>

     void
     _LIBC_EfiExit(EFI_STATUS Status, UINTN DataSize, CHAR16, *ExitData)

DESCRIPTION

     _LIBC_EfiExit() terminates a process and retuns EFI specific exit
     parameters to the parent.  As with exit(3), it performs the following
     functions in the order listed:

	   1.	Call the functions registered with the atexit(3) function, in
		the reverse order of their registration.

	   2.	Flush all open output streams.

	   3.	Close all open streams.

     Unlike exit(), it allows a full EFI_STATUS type (as opposed to int) and
     exit data to be returned to the parent.

RETURN VALUES

     The _LIBC_EfiExit() function never returns.
