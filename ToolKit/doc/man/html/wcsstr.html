NAME

     wcsstr - locate a substring in a wide string

SYNOPSIS

     #include <string.h>

     wchar_t *
     wcsstr(const wchar_t *big, const wchar_t *little)

DESCRIPTION

     The wcsstr() function locates the first occurrence of the null-terminated
     string little in the null-terminated string big. If little is the empty
     string, wcsstr() returns big; if little occurs nowhere in big, wcsstr()
     returns NULL; otherwise wcsstr() returns a pointer to the first character
     of the first occurrence of little.
