NAME

     wcsspn - span a wide string

SYNOPSIS

     #include <string.h>

     size_t
     wcsspn(const wchar_t *s, const wchar_t charset)

DESCRIPTION

     The wcsspn() function spans the initial part of the null-terminated
     string s as long as the characters from s occur in string charset.

RETURN VALUES

     The wcsspn() function returns the number of characters spanned.
