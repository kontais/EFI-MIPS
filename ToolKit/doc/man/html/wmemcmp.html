NAME

     wmemcmp - compare wchar_t string

SYNOPSIS

     #include <string.h>

     int
     wmemcmp(const void *b1, const void *b2, size_t len)

DESCRIPTION

     The wmemcmp() function compares wchar_t string b1 against wchar_t string b2.
     Both strings are assumed to be len wchar_t's long.

RETURN VALUES

     The wmemcmp() function returns zero if the two strings are identical, oth-
     erwise returns the difference between the first two differing wchar_ts.  Zero-length
     strings are always identical.
