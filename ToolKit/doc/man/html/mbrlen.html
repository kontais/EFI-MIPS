NAME

     mbrlen - gets length of a multibyte character string

SYNOPSIS

     #include <wchar.h>

     size_t
     mbrlen ( const char * s ,  size_t n  ,  mbstate_t * pst  );

DESCRIPTION

     The mbrlen() function is equivalent to the call: mbrtowc(NULL, s, n, ps
     != NULL ? ps : &internal) where internal is the mbstate_t object for the
     mbrlen function, except that the expression designated by ps is evaluated
     only once.ate.

RETURN VALUES

     The mbrlen() function returns a value between zero and n, inclusive,
     (size_t)-2,or (size_t)-1.
     This function has not been actually implemented. So it will always return -1.

SEE ALSO

     btowc(3),  wctob(3),  mbsinit(3),  mbrtowc(3),  wcrtomb(3),

STANDARDS

     The mbrlen() function conforms to ISO 9899: 1990 (``ISO C'').
