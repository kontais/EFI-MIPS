NAME

     wcscat - concatenate wide strings

SYNOPSIS

     #include <string.h>

     wchar_t *
     wcscat(wchar_t *s, const wchar_t *append)

     wchar_t *
     wcsncat(wchar_t *s, const wchar_t *append, size_t count)

DESCRIPTION

     The wcscat() and wcsncat() functions append a copy of the null-terminated
     string append to the end of the null-terminated string s, then add a ter-
     minating `\0'. The string s must have sufficient space to hold the re-
     sult.

     The wcsncat() function appends not more than count characters from
     append, and then adds a terminating `\0.'

RETURN VALUES

     The wcscat() and wcsncat() functions return the pointer s.
