NAME

     wctrans - Determines a mapping from one set of wide-character codes to another.

SYNOPSIS

     #include <wctype.h>

     wctrans_t 
     wctrans(const wchar_t *property)

DESCRIPTION

     wctrans This function determines a mapping from one set of wide-character codes to another.

RETURN VALUES

     The function returns the transformation of the character c,
     using the transform in category.

SEE ALSO

     wctrans(3)

STANDARDS

     The towctrans() function conforms to ISO 9899: 1990 (``ISO C'').
