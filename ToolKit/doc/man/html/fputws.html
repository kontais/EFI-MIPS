NAME

     fputws, putws - output a line to a stream

SYNOPSIS

     #include <wchar.h>

     int
     fputws(const wchar_t *str, FILE *stream)

     int
     puts(const wchar_t *str)

DESCRIPTION

     The function fputws() writes the wide character string pointed to by str 
     to the stream pointed to by stream.

     The function putws() writes the wide character string str, and a terminating
     newline character, to the stream stdout.

RETURN VALUES

     The fputws() function returns 0 on success and WEOF on error; putws() re-
     turns a nonnegative integer on success and EOF on error.

ERRORS

     [EBADF]  The stream supplied is not a writable stream.

     The functions fputws() and putws() may also fail and set errno for any of
     the errors specified for the routines write(2).

SEE ALSO

     ferror(3),  putwc(3),  stdio(3)

STANDARDS

     The functions fputws() and putws() conform to ISO 9899: 1990 (``ISO C'').
