NAME

     wmemchr - locate wchar_t in wchar_t string

SYNOPSIS

     #include <string.h>

     void *
     wmemchr(const void *b, int c, size_t len)

DESCRIPTION

     The wmemchr() function locates the first occurrence of c in string b.

RETURN VALUES

     The wmemchr() function returns a pointer to the wchar_t located, or NULL if
     no such wchar_t exists within len wchar_t's.
